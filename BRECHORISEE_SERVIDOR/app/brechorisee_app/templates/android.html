Android sem Android Studio

App BRECHORISEE conectado ao banco local do sistema.

O banco de dados principal continua local no computador, no arquivo SQLite do sistema. O app Android abre o mesmo sistema pela rede Wi‑Fi, então o cadastro, fotos, caixa, vendas, estoque e relatórios ficam interligados com o computador.

📱 app Android nativo com WebView 🗄️ usa o banco local do computador 📸 câmera e envio de fotos pelo celular 🔎 busca digitada, QR/código e reconhecimento 🎙️ voz nativa do Android para preencher campos ☁️ sem nuvem obrigatória

1. Ligue o servidor local

No computador, abra ABRIR_SERVIDOR_PARA_APP_ANDROID.bat ou INICIAR_PARA_CELULAR.bat.

2. Compile o APK por .bat

Na pasta brechorisee_android, dê dois cliques em COMPILAR_APK_WINDOWS.bat. Não precisa abrir Android Studio.

3. Instale no celular

Copie o arquivo BRECHORISEE_android.apk para o Android ou use INSTALAR_NO_CELULAR_USB.bat. Ao usar 🎙️, o app pedirá permissão de microfone.

4. Configure uma vez

Ao abrir o app, informe o endereço {{ phone_url }}. Depois o app lembra esse endereço.

Como o banco fica interligado

Um banco local, dois acessos

O arquivo do banco fica em: {{ db_path }}. O computador roda o sistema local e o app Android acessa esse mesmo sistema pela rede. Assim, quando você cadastra uma peça pelo celular, ela aparece no computador; quando vende no caixa, a baixa também vale para os dois.

Se o computador estiver desligado, fora do Wi‑Fi ou bloqueado pelo firewall, o app não consegue acessar o banco. Nesse caso, abra o servidor local e libere a porta 8000.
