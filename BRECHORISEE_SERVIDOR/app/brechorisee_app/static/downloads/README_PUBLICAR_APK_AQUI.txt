Coloque aqui o APK cliente com o nome BRECHORISEE_CLIENTE.apk.

Quando o servidor estiver ligado, as clientes poderão baixar por:
http://IP_DO_SERVIDOR:8000/download/app-cliente.apk
http://IP_DO_SERVIDOR:8000/apk
http://IP_DO_SERVIDOR:8000/app/cliente

No Windows, use o botão "Publicar APK Cliente" no programa BRECHORISEE_WINDOWS.
