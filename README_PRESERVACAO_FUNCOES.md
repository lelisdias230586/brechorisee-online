# Preservação das funções da BRECHORISEE

Esta versão simplifica a instalação, mas não remove a base principal.

Mantido:

- `BRECHORISEE_SERVIDOR\app\brechorisee_app\app.py`
- `templates`
- `static`
- apps Android cliente/admin
- projetos iOS cliente/admin
- funções de live, admin, cliente, carrinho, pedidos, Telegram, bot e downloads
- banco SQLite local
- backup local

Simplificado:

- um único `.env`
- um único BAT principal
- Central BRECHORISEE com botões
- backup manual e automático
- QR Code para APK
- links automáticos por IP local
- atalhos na Área de Trabalho

O caminho principal agora é local:

```text
Notebook Windows ou celular Android servidor
```

Render/GitHub deixam de ser obrigatórios para uso diário, mas os arquivos antigos de apoio permanecem preservados quando existirem.
