# BRECHORISEE Admin — iOS

Este é o projeto base iOS para abrir o BRECHORISEE na nuvem.

## URL inicial
`https://brechorisee-online.onrender.com/admin-acesso`

## Como compilar
Para gerar app iOS real é necessário:
- Mac
- Xcode
- Conta Apple Developer para assinar e distribuir
- Bundle ID: `com.brechorisee.admin`

## Alternativa sem App Store
Use o PWA pelo Safari:
1. Abra `https://brechorisee-online.onrender.com/admin-acesso` no Safari do iPhone.
2. Toque em Compartilhar.
3. Toque em "Adicionar à Tela de Início".

## Segurança
- Admin abre área administrativa.
- Cliente abre somente área da cliente/vitrine.
