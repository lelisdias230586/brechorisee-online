# BRECHORISEE Cliente Android

Pasta do app cliente Android.

## Gerar AAB/APK no Windows

Entre em:

```text
BRECHORISEE_CLIENTE\android
```

Rode:

```bat
GERAR_PUBLICACAO_COMPLETA_WINDOWS.bat
```

O APK para download local é copiado para o servidor em:

```text
BRECHORISEE_SERVIDOR\app\brechorisee_app\static\downloads
```

Guarde a pasta `play_store_upload` criada durante o build. Ela contém a chave de publicação do app.
