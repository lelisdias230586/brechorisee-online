- Padronização de nome dos arquivos para "Sistema BRECHORISEE".
- Pacote completo, não apenas patch.
- Um comando principal no Windows: `SISTEMA_BRECHORISEE.cmd`.
- Um comando principal no celular/Termux: `SISTEMA_BRECHORISEE_CELULAR.sh`.
- PC configurado para acessar o servidor oficial do celular: `http://192.168.1.18:8000`.
- Atalhos do PC deixam de usar `127.0.0.1`.
- `.env` ajustado para servidor oficial do celular.
- Mantidos módulos Cliente, Admin, Servidor, Android servidor, iOS e Windows.
- Mantidas correções anteriores: rede externa, configuração inicial dos apps, cards/overlay, Admin modo seguro e download do APK.

Observação:
- O Android não permite script trocar IP do Wi-Fi sem root. O IP fixo deve ser configurado no Android/roteador uma vez.
